# vue-template

## Usage

这是基于 [vue-cli 3.0](https://cli.vuejs.org/zh/) 的一个项目模版.

``` bash
npm install -g (yarn global add) @vue/cli
vue init LiuCaiHe/vue-template <project name>
cd <project name>
yarn
yarn run serve
```
