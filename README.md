# vue-template

## Usage

这是基于 [vue-cli 3.0](https://cli.vuejs.org/zh/) 的一个项目模版.

``` shell
# 全局安装 vue-cli 3.0
$ yarn global add @vue/cli

# 初始化项目 <LiuCaiHe/vue-template> 项目模版
$ vue init LiuCaiHe/vue-template <project name>

# 进入目录
$ cd <project name>

# 运行
$ yarn run serve
```

## What's Included

``` shell
# 启动本地服务
$ yarn run serve

# 预览
$ yarn run preview

# 生产
$ yarn run build

```
